**화면 목록 (web 스파이크 구현 순서)**
1. 웰컴/온보딩(닉네임) — *index.html 내 오버레이, 최초 1회*
2. 홈 허브 — *index.html (재설계 중심)*
3. 모드 인트로 — *각 모드 페이지 상단*
4. 세션 — *listening.html (재스타일)*
5. 결과/축하 — *listening.html 결과화면 (컨페티·마스코트)*
6. 진정 인터루드 — *구현 완료, 디자인만 통일*
7. 통계 / 8. 프로필 — *후속*

**원칙**: 모든 세션은 "짧게(≤5분) → 즉시 보상 → 숨 고르기 → 홈"으로 닫힌 루프. 좌절 없는 80% 난이도(끈기), 안전한 실패(두려움), 마무리 호흡(심리안정)이 흐름 자체에 박혀 있음.

## 1.5 실제 레퍼런스 DNA (Mobbin 로그인 후 실측, 2026-06-10)

Duolingo ABC 실제 화면 3종(Onboarding 스플래시·Home·Level detail)을 직접 확인. **어림짐작 아님.**
- **배경**: 풀블리드 소프트 그라데이션(보라/핑크) 또는 따뜻한 우드 디오라마. **흰 배경 없음.**
- **마스코트 중심**: 온보딩·홈 모두 큰 둥근 캐릭터가 주인공. 홈은 마스코트가 사는 살아있는 세계.
- **원형 아이콘 버튼**: 흰 원 + 컬러 글리프 + 소프트 섀도 + 알림닷 (상단 내비/액션).
- **진행**: 둥근 알약 바 + 별 + "0/13".
- **워드마크**: 청키 라운드 멀티컬러("duolingo abc" a파랑/b빨강/c노랑).
- **오디오 우선**: 읽기 화면에 스피커 아이콘 → ESC 정체성과 일치.

**ESC 적응(중요)**: Duo의 핵심은 맞춤 일러스트/애니 디오라마 = 대형 아트 투자. 웹 스파이크는 1:1 복제 불가.
→ **CSS 재현 가능 언어**(그라데이션 배경·원형 버튼·3D 버튼·알약 진행바·멀티컬러 워드마크) + **SVG 마스코트 1종(에코🎧)** + Kenney로 80% 확보. 디오라마는 추후 아트 예산.

## 2. 디자인 시스템 (토큰)

### 폰트 (Google Fonts, 무료, <link>/@import)
- **디스플레이/제목 = `Jua`** — 둥글고 친근한 한글 (Duolingo ABC 톤).
- **본문 = `Noto Sans KR`** (400/500/700/900).
- **숫자/강조 = `Fredoka`** — 통통한 라틴 숫자 (스트릭·레벨·점수).

### 컬러 — ❌흰배경+보라 금지 / ✅따뜻한 크림 + 코럴
```
--cream    #FFF7EC   (배경, 흰색 대신)
--cream-2  #FFEFD9
--ink      #3A2E25   (따뜻한 다크브라운, 검정 대신)
--ink-soft #8A7868
--line     #ECDDC6   (따뜻한 보더/셸프)
모드 컬러코드 (각 base/shelf):
  듣기 coral  #FF7A4D / #E2602F   ← 브랜드 메인(히어로)
  집중 blue   #4D9BFF / #2F7AD8
  속도 amber  #FFB52E / #E89512
  도전 violet #9B7BFF / #7C5BE0
  끈기 green  #5BD06B / #3FB04F
  진정 teal   #2FC8B4 / #1FA897
```

### 시그니처 컴포넌트 — **3D 눌리는 버튼** (Duolingo의 핵심 촉감)
- 둥근 사각(radius 16) + 아래 더 진한 "shelf"(box-shadow 0 5px 0 dark) + 흰 Jua 텍스트.
- `:active` → `translateY(5px)` + shelf 제거 = 물리적으로 눌리는 느낌. **이 디테일 하나가 "앱답다"를 만든다.**

### 카드/타일
- 흰 카드 + 3px 따뜻한 보더 + 아래 4px shelf 그림자 + radius 22. 모드 타일은 모드 컬러 보더.

### 마스코트
- **헤드폰 쓴 둥근 캐릭터**(듣기·집중 앱 정체성). 인라인 SVG. 웰컴/홈/결과에 등장, 반응(idle bob, 정답시 점프).

### 모션 (CSS-only)
- 로드: 카드 staggered 등장(스프링). 버튼: press depress. 마스코트: idle bob. 스트릭: 불꽃 흔들. 결과: 컨페티.

### 배경/질감
- 크림 + 아주 옅은 따뜻한 도트/블롭 (보라 radial 폐기).

### 에셋 (CC0 / 무료)
- **Kenney**: UI Pack(버튼·패널), Game Icons, Input Prompts, Interface Sounds·UI Audio(탭·정답·축하음), Particle Pack(컨페티), Toon/Animal(마스코트 보강). 전부 CC0.
- **사운드**: Kenney UI Audio로 탭/정답/오답/완료음 (절제해서).
- **이모지**: 필요시 Noto Emoji(구글, 무료).

### 비의료 가드 유지
- 톤은 놀이·격려. "치료/진단/증상" 금지. 마스코트도 의료 아닌 학습 친구.
